# noobpack

webpack2 config template

most features are optional, and can be combined based on `--env` flags

see package.json for usage

includes;
    tree shaking
    SASS / ES6 / stage-0
    CSS extract
    CSS modules
    ESLint
    babili
    closure-compiler
    webpack-dashboard
